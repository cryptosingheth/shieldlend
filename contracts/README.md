# contracts/

Solidity smart contracts for ShieldLend. Built and tested with Foundry.

## Planned Contracts

| File | Status | Description |
|------|--------|-------------|
| `src/ShieldedPool.sol` | 🔜 Step 5 | Incremental Merkle tree of commitments; handles deposit + withdraw |
| `src/NullifierRegistry.sol` | 🔜 Step 5 | Tracks spent nullifiers to prevent double-withdrawal |
| `src/LendingPool.sol` | 🔜 Step 8 | Aave V3 fork — adds borrow/repay on top of shielded pool |
| `src/DepositVerifier.sol` | 🔜 Step 4 | Auto-generated by snarkjs from deposit.circom |
| `src/WithdrawVerifier.sol` | 🔜 Step 4 | Auto-generated by snarkjs from withdraw.circom |
| `src/CollateralVerifier.sol` | 🔜 Step 4 | Auto-generated by snarkjs from collateral.circom |

## Commands

```bash
# Build
forge build

# Test
forge test

# Test with verbosity
forge test -vvvv

# Deploy (testnet)
forge script script/Deploy.s.sol --rpc-url $RPC_URL --broadcast
```

## Environment Variables

```bash
RPC_URL=<Horizen L3 testnet RPC or Base Sepolia RPC>
PRIVATE_KEY=<deployer private key>
```

See [`docs/architecture.md`](../docs/architecture.md) for full contract interface definitions.
